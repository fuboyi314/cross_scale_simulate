# AGENTS.md

## Project
本仓库用于开发科研软件：
《基于LBM的跨尺度多孔介质渗流模拟平台》

## V1.0 scope
- 仅二维
- 仅单相
- 仅稳态
- 仅不可压近似
- 数值方法采用 D2Q9 + BGK/SRT
- 首版先完成 GUI、项目管理、图像导入、预处理骨架、日志、结果面板
- 首轮不要实现完整三维、热耦合、多相流、THMC

## Engineering rules
- 使用 Python
- 使用 PySide6
- 严禁把项目写成单文件大脚本
- GUI、算法、预处理、日志、导出、项目管理必须解耦
- 所有核心类需要 type hints 和 docstrings
- 不要用大量 TODO 代替真实实现
- 不要硬编码个人路径
- 默认兼容 Windows；若涉及 CLI 工作流，优先考虑 WSL 工作区
- 每完成一个任务，必须说明：
  - 新增/修改了哪些文件
  - 如何运行
  - 哪些已完成
  - 哪些仍为占位

## GUI rules
- 图形界面是一等核心需求，不是附属功能
- 主窗口必须包括：
  - 菜单栏
  - 工具栏
  - 左侧参数面板
  - 中央标签式显示区
  - 右侧结果摘要面板
  - 底部日志面板
  - 状态栏
- 所有核心功能必须能从 GUI 触发
- 长耗时任务不得阻塞界面
- GUI 风格应接近科研/工程软件，而不是演示玩具

## First-round task priority
第一轮只优先完成：
1. 项目目录骨架
2. app.py 入口
3. 主窗口与主要 widget 骨架
4. 项目管理模块
5. 日志系统
6. GUI 动作连接
7. 图像导入基础能力
8. 预处理接口占位与可扩展结构

## Done criteria for first round
- 软件可以启动
- 可以新建/打开/保存项目
- 可以导入二维图像并预览
- GUI 各区域完整
- 日志能同时写到界面和文件
- 未完成按钮也有明确反馈，不是空动作
